..moduleauther:: Fangyi

class nilm.trainer.trainer.Trainer(config, users='all')

訓練 nilm 模型的訓練 pipeline

參數

  • config (dictionary) – 訓練參數

  • users (list or str) – 使用者ID。example: [144660, 14665]。all：自動搜尋全部。

apps_to_train()

確認屬性 apps 之順序是否正確，總電量需要擺在第一順位

傳回

a list

datapipleline(start, end, csv, npy)

資料取樣的 pipeline

參數

  • start (str, YYYY-mm-dd) – 資料區間之起始點。example: 2017-09-01

  • end (str, YYYY-mm-dd) – 資料區間之最終點。example: 2017-09-30

  • csv (str) – 讀取 raw_csv之路徑。default:./data/DSM/raw_dataset.csv

  • npy (str) – 存取資料清理後之路徑，為 npy 檔案。default:./data/DSM/pre_from_csv.csv

傳回

the RealandSynSample class

to_train_model(datapipeline_for_train, datapipeline_for_test)

初始化模型或者讀取模型，並執行相關模型任務

參數

  • datapipeline_for_train (class) – 訓練資料之資料取樣 pipeline

  • datapipeline_for_test (class) – 測試資料之資料取樣 pipeline

train()

啟動深度學習模型訓練的 pipeline
